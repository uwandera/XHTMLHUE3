Documentação do Projeto Web
Visão Geral
Este repositório contém um projeto web desenvolvido com HTML5 e CSS3, com foco em organização estrutural, responsividade e boas práticas de front-end. O projeto é composto por múltiplas páginas estáticas, um arquivo central de estilos e uma estrutura preparada para futura expansão com JavaScript.

1. Como Usar o Projeto (Getting Started)
1.1 Pré-requisitos
Para execução local do projeto, é necessário:

Navegador web moderno (Google Chrome, Mozilla Firefox, Microsoft Edge ou Safari)
Editor de código-fonte (recomenda-se Visual Studio Code)
Servidor web local (ex.: Live Server, XAMPP, Python ou Node.js)


1.2 Instalação e Execução

Clone o repositório:

git clone https://github.com/seu-usuario/seu-repositorio.git
cd seu-repositorio

Inicie um servidor local, utilizando uma das alternativas abaixo:

Opção 1 – Live Server (VS Code)
Utilize a extensão Live Server e clique com o botão direito em index.html, selecionando Open with Live Server.
Opção 2 – Python
python -m http.server 8000
npx http-server

Acesse a aplicação pelo navegador:


http://localhost:8000
ou http://localhost:5500 (padrão do Live Server)

2. Funcionalidades do Projeto

Layout responsivo compatível com dispositivos móveis, tablets e desktops
Navegação entre páginas por meio de menu estruturado
Formulários com múltiplos tipos de entrada
Interface visual moderna com uso de gradientes, sombras e transições
Aplicação de boas práticas de acessibilidade (labels, contraste e hierarquia visual)
Centralização de estilos em CSS externo
Componentes reutilizáveis baseados em classes CSS
Estrutura preparada para inclusão futura de interatividade em JavaScript

6. Guia de Estilo (Style Guide)
6.1 Tipografia
<h1>Título principal (36px)</h1>
<h2>Subtítulo (28px)</h2>
<h3>Seção (22px)</h3>
<h4>Subseção (18px)</h4>

Tamanhos de fonte

Corpo do texto: 16px
Labels: 14px
Texto pequeno: 12px
Destaque: 18px ou superior


6.2 Espaçamentos
